O que são Atributos?

Atributos são informações que passamos na Tag para que ela se comporte da maneira esperada. Existem atributos globais (que funcionam em todas as Tags) e específicos (que são direcionados para cada Tag, através de especificação).

Atributos globais

Seguem alguns atributos globais importantes e suas descrições simplificadas:

  1. Title

    2. Representa um auxílio ao elemento. Semelhante a dica do elemento.

  2. Style

    3. Determina propriedades CSS diretamente no elemento.

  3. Class

    4. Determina uma (ou mais) classe(s) para o elemento. Para definir mais de uma classe, coloque-as separadas por espaço.

  4. Id

    5. É o identificador único do elemento. Somente deve ser declarado um id por elemento. E este id não deve ser repetido na mesma página.

  5. Lang

    6. Determina o idioma em que está escrito o conteúdo do elemento.

Tag Mark

A tag mark é usada para destacar ou realçar um trecho de texto no conteúdo da página. Ela pode ser usada para chamar a atenção do usuário para informações importantes ou para destacar termos de busca.

Exemplo:

O evento acontecerá no dia 10 de maio no centro de convenções da cidade.

Tag hr

A tag hr é usada para inserir uma linha horizontal na página, geralmente para separar seções de conteúdo. Ela não tem conteúdo ou valor, e é usada apenas para fins de formatação.

Exemplo:

Este é um parágrafo.

Este é outro parágrafo.

Tag Sub

A tag sub é usada para criar um subscrito, que é um texto menor e posicionado abaixo da linha de base do texto adjacente. Ela pode ser usada para indicar uma fórmula química, uma nota de rodapé, ou qualquer outra informação que precise ser exibida em uma posição inferior.

Exemplo:

H2O

Tag Sup

A tag sup é usada para criar um sobrescrito, que é um texto menor e posicionado acima da linha de base do texto adjacente. Ela pode ser usada para indicar uma potência, uma nota de rodapé, ou qualquer outra informação que precise ser exibida em uma posição superior.

Exemplo:

x2 + y2 = r2

Tag Blockquote

A tag blockquote é usada para citar uma seção de texto de outra fonte, como um trecho de uma entrevista ou de um artigo. Ela é geralmente exibida com um recuo na margem esquerda e/ou direita, e pode ter um estilo visual distinto para diferenciá-la do restante do conteúdo.

Exmeplo:

“A verdadeira viagem do descobrimento não consiste em procurar novas paisagens, mas em ter novos olhos.”

Marcel Proust

Tag Font

A tag font foi usada em versões antigas do HTML para especificar a fonte, a cor e o tamanho do texto. No entanto, ela foi considerada obsoleta e não deve ser mais usada. Em vez disso, recomenda-se o uso de folhas de estilo CSS para estilizar o conteúdo HTML.

Tag Del

A tag del é usada em HTML para indicar que um trecho de texto foi removido ou excluído de uma versão anterior de um documento. Ela é frequentemente usada em documentos colaborativos para mostrar as alterações que foram feitas em um texto ao longo do tempo.

Exemplo:

O preço original do produto é R$ 100,00 R$ 80,00.

Tag abbr

A tag abbr é usada em HTML para definir uma abreviatura ou acrônimo. Ela é frequentemente usada em documentos técnicos ou acadêmicos para fornecer uma explicação para um termo que pode não ser conhecido por todos os leitores.

Exemplo:

A missão da DIO é criar novas possibilidades para pessoas e organizações de todos os lugares.

(Voltar)
