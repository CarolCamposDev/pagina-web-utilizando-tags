Lista Ordenada

Lista ordenada é uma lista de itens que são numerados em uma sequência ordenada. Por padrão, os números são usados como marcadores para cada item da lista.

Exemplo:

  1. Item 1
  2. Item 2
  3. Item 3

Lista Não Ordenada

Lista não ordenada é uma lista de itens que não são numerados ou classificados em uma ordem específica. Em vez disso, cada item é representado por um marcador padrão, como um ponto.

Exemplo:

  • Item 1
  • Item 2
  • Item 3
(Voltar)
